Use voice-prices in your code

Install the package and compute costs in a few lines.

Install
uv add voice-prices # or: pip install voice-prices
Price an LLM call
from voice_prices import Usage, calc_price price = calc_price(Usage(input_tokens=1000, output_tokens=200), 'gpt-4o') print(price.total_price)
Price a voice model through LiveKit
from voice_prices import Usage, calc_price # one minute of Deepgram Nova-2 STT, priced via LiveKit Inference price = calc_price(Usage(audio_input_seconds=60), 'deepgram/nova-2', provider_id='livekit') print(price.total_price)
